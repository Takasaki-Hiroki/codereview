テストケース一覧

Cardクラスのテストケース
1. getMarkでそのカードのマークが取得できること
2. getNumberでそのカードの番号が取得できること
3. getIdでそのカードのidが取得できること
4. isJokerでジョーカーであるかを取得できること
5. カードに順序があること
6. カードがソートできること
7. 不適切なカードを生成できないこと1 @例外
8. 不適切なカードを生成できないこと2 @例外

Deckクラスのテストケース
1. getNumCards()で現在のカード枚数を取得できること
2. deck生成時に52枚とジョーカ分のカードを保持していること
3. deckから指定枚数のカードを引けること
4. デッキがランダムになっていること ->　目視確認しか思いつかなかった
5. 残り枚数以上のカードを引けないこと @例外
6. ゼロ以下の整数の枚数分カードを引けないこと @例外
7. 負の数の枚数分ジョーカがあるデッキを作れないこと @例外
